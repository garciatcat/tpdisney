POSIBLES ERRORES

Código de error 83: “Algo salió mal. Intenta de nuevo...”.


Causa del error:


Por lo general, se debe a un problema de compatibilidad o a un error de conexión. Estas son algunas de las causas posibles: Acceso a Disney+ con un método o dispositivo no compatible (p. ej., un emulador).

Ejemplo: ver un video en el navegador web del dispositivo en vez de usar la aplicación de Disney+. Conexión a internet lenta o inestable.


Solución del problema


Para resolver este y otros problemas relacionados, puedes probar lo siguiente:

Asegúrate de que el dispositivo en el que inicias la aplicación de Disney+ o la computadora en la que accedes a DisneyPlus.com sean compatibles.

Verifica la conexión a internet.

Reinicia la aplicación de Disney+ o actualiza el sitio. Luego, espera unos minutos e intenta ver el video de nuevo.

Apaga y enciende el dispositivo, y luego reinicia la aplicación de Disney+.

Si tienes inconvenientes para iniciar sesión después de reiniciar el dispositivo, consulta Problemas para iniciar sesión en Disney+.

Si no logras resolver el problema con los consejos anteriores, prueba estas otras soluciones.

Si el problema continúa, contáctanos directamente para que podamos ayudarte.



Código de error 39: “Lo sentimos, no se puede mostrar el video...”.


Causa del error:


Esto suele suceder cuando el contenido solicitado está temporalmente inaccesible. Estas son algunas de las causas posibles:

Una conexión HDMI (HDCP) que no funciona o no es compatible*.

Quieres acceder a Disney+ desde un dispositivo o computadora no compatibles.

Conexión a internet o señal wifi inestables.

Número inusualmente elevado de solicitudes simultáneas de un contenido específico.


Solución del problema


Para resolver este y otros problemas relacionados, puedes probar lo siguiente:

Desconecta y vuelve a conectar ambos extremos del cable HDMI conectado al dispositivo.

Usa un puerto HDMI diferente en la TV.

Usa otro cable HDMI.

Asegúrate de que el dispositivo en el que inicias la aplicación de Disney+ o la computadora en la que accedes a DisneyPlus.com sean compatibles.

Asegúrate de que la aplicación o navegador tenga instaladas las últimas actualizaciones.

Verifica la conexión a internet.

Reinicia la aplicación de Disney+ o actualiza el sitio. Luego, espera unos minutos e intenta ver el video de nuevo.



Si no lo puedes solucionar:

Comunícate con Atención al Cliente de Disney+. Escribe tu nombre y el correo electrónico de la cuenta. Pueden aplicarse cargos locales o internacionales.
